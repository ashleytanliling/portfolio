Photo Album

An Express application to upload / edit / delete photos & add / delete reviews belonging to you.

TL;DR - Authentication, Authorization, Cloudinary, MapBox, Deployment

  • Front-end : EJS, Bootstrap styling, flash messages, stars rating for reviews

  • Database : MongoDB & MongoDB Atlas

  • Data modeling : Photo and Reviews (1-to-many relationship)

  • Data validation : Bootstrap (front-end) & Joi (back-end)

  • Passport.js : Register users, login, logout (Authentication required to add photos & reviews)

  • Authorization : Users can only delete photos & reviews belonging to them

  • Cloudinary : image storage & transformation

  • MapBox : forward geocoding & cluster map

  • Simple Security measures

  • Render : deployment

Project takes a while to load. Appreciate your patience. Thank you! 😊
Hope you enjoy viewing it as much as I enjoyed building it!
